Basic informations about CSS Styleheets

Cascading style sheets (CSS) - are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.

CSS - is the language for describing the presentation of Web pages, including colors, layout, and fonts. It allows one to adapt the presentation to different types of devices, such as large screens, small screens, or printers. CSS is independent of HTML and can be used with any XML-based markup language.

CSS - Cascade Style sheets are used for giving a better look to the web pages.

CSS - are using selectors to edit singular or multiply elements of HTML code

CSS - simply is used to edit styles defaulted by the web browsers.

CSS - files which are recommended to use, are linked to the HTML code by link tag which should be placed in the head of the HTML code

CSS - as was marked above is using selectors as id or class as same as the tag names etc.

CSS - can be commented by using /* Here you can enter comment of CSS code */ comment marks
